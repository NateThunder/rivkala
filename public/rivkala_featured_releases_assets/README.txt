Rivkala Featured Releases Asset Pack

Folders:
- backgrounds: cream paper background, black paper texture, black torn top edge.
- decor: lamp cutout, surreal eyes/lips collage, tape, torn paper strips, scraps, album card mat.
- releases: cropped album cover images from the supplied release screenshot.
- example: simple HTML and CSS starter for the section.

Suggested usage:
1. Put the folders inside your site's public/assets/rivkala-featured-releases/ folder.
2. Update the CSS image paths if your folder location is different.
3. Replace the cropped release covers with higher-resolution final cover images when you have them.

Note:
The mockup image is included as featured-releases-mockup.png for visual reference, but build the section from the separated assets for a proper responsive website.
